# Literature Review Workflow

## Daily Reading Loop

1. Pick one note from [[Metal Alloy Processing for In-Space Manufacturing MOC]].
2. Open the linked source PDF from `ReferencePapers`.
3. Add 3-5 thesis-relevant numerical values to the paper note.
4. Link each value to a concept note such as [[Columnar-to-Equiaxed Transition]] or [[Interdendritic Flow and Macrosegregation]].
5. Add one sentence to [[Earth 1g vs Microgravity Solidification Processing]] only if it changes the synthesis.

## Paper Note Rule

Each paper note should answer:

- What system was studied?
- What was the gravity condition?
- What was measured numerically?
- What changed between 1g and microgravity?
- Why does it matter for in-space manufacturing?

## Concept Note Rule

Each concept note should answer:

- What is the mechanism?
- What equation or scaling controls it?
- How does gravity change it?
- Which papers provide the strongest evidence?

## Weekly Synthesis Loop

Every week, update:

- [[Metal Alloy Processing for In-Space Manufacturing MOC]]
- [[Earth 1g vs Microgravity Solidification Processing]]
- one open question in the relevant concept note
